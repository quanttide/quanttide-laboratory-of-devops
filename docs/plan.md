# plan

扫描项目管理文件，生成规划摘要。

## 用法

```bash
qtcloud-devops-code plan
```

## 输出

```
项目规划摘要
----------------------------------------
  BUGS.md: 不存在
  ROADMAP.md (迭代计划): 42 行
  TODO.md (待办事项): 54 行
    待办: 25 完成: 0
  CHANGELOG.md (变更记录): 24 行

  总计: 4 文件, 120 行
  待办项: 25
```

### 扫描文件

| 文件 | 说明 |
|------|------|
| `BUGS.md` | 已知缺陷列表 |
| `ROADMAP.md` | 迭代计划与优先级 |
| `TODO.md` | 待办事项清单 |
| `CHANGELOG.md` | 版本变更记录 |

不存在的文件会显示"不存在"，不报错。
